# Projeto de Algoritmos - UnB

## Introdução

A disciplina de **Projeto de Algoritmos**, na *Universidade de Brasília (UnB/FCTE)*, foi dividida em 5 módulos para a abordagem dos [conteúdos](#conteúdo-da-disciplina) da base curricular. Em cada módulo, os alunos deviam se organizar em duplas para o desenvolvimento de um trabalho, no [*GitHub*](https://github.com/projeto-de-algoritmos-2025), que pusesse em prática um dos conteúdos daquele módulo. Esse repositório tem como objetivo guardar os 5 trabalhos desenvolvidos pela [**Dupla 30**](#dupla-30) nessa disciplina ao longo do semestre.

<br>

## Conteúdo da Disciplina

> **Grafos 1**
> - *BFS e DFS*;
> - *Flood Fill*;
> - *Componentes Conectados e Conectividade*;
> - *Bipartição*;
> - *Ordenação Topológica*;
> - *Componentes Fortemente Conectados*.

> **Grafos 2**
> - *Dijkstra*;
> - *Prim*;
> - *Union-Find e Kruskal*.

> **Algoritmos Ambiciosos**
> - *Interval Scheduling*;
> - *Interval Partitioning*;
> - *Scheduling to Minimize Lateness*;
> - *Knapsack*;
> - *Huffman Codes*.

> **Dividir e Conquistar**
> - *Contagem de Inversões*;
> - *Mediana das Medianas*;
> - *Par de Pontos Mais Próximos*.

> **Programação Dinâmica**
> - *Weighted Interval Scheduling*;
> - *Weighted Knapasack*;
> - *Alinhamento de Sequências*;
> - *Bellman-Ford*.

# Execução

1. Clone o repositório:

```
git clone https://github.com/eduardodpms/UnB_PA
```

2. Acesse a pasta do repositório:

```
cd UnB_PA
```

3. Execute o arquivo `run.py`, passando como argumento o número do projeto que deseja executar (1-5). Exemplo para o primeiro projeto:

```
python run.py 1
```

***Obs:** Alternativamente, você pode acessar a pasta de cada um dos projetos e executar o arquivo `main.py`.*

<br>

# Requisitos

Todos os projetos aqui foram desenvolvidos em `Python`, utilizando bibliotecas nativas da linguagem, como o `tkinter` e o `math`. Adicionalmente, o **Projeto 3** usa a biblioteca `bitarray`. Para instalá-la, utilize:

```
pip install bitarray
```
